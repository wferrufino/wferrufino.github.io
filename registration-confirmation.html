Quick Start Guide

Log & Event Manager normalizes logs so your rules and reports work regardless of the source. For example, see all logon failures regardless of the original log structure. Follow these steps to install your trial and begin monitoring in about an hour.

  • Import The Virtual Appliance
  • Assign IP Address to The VM

    Before you begin, make sure you have the baseline system requirements for the evaluation installation:

    • Windows Server ® 2008/2012/2014 OS
    • 4-8GB RAM
    • 2-4 Dual-core processors
    • VM acceptable
    There are two installation options:
    • Express: automatically installs a free version of MS SQL
    • Express with a database size limitation of 10GB
